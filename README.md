### Installation

Requires Node 12.13.1^ & npm 6.12.1^ to run.

• Clone the repo
```
 $ git clone https://github.com/Supra-MS/You-Dont-Know-JS_Quizzes.git
```
• From within the root directory run,
```
 $ npm install
```
---

### Local development

First seed the db using,
```
$ npm run seed
```

To run  the server,
```
$ npm start
```

To build the client,
```
$ npm run build

or

$ npm run dev:build
